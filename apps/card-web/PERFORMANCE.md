```powershell

```

## 优化前后

| 指标 | 优化前 | 优化后 | 变化 |
| --- | ---: | ---: | ---: |
| 首屏入口 JS | 543.24 kB | 305.21 kB | -43.81% |
| 首屏入口 JS gzip | 166.38 kB | 91.56 kB | -44.97% |
| CSS | 63.01 kB | 64.28 kB | +2.02% |
| CSS gzip | 12.30 kB | 12.52 kB | +1.79% |

优化后，`index.html` 只加载入口，不预加载 AI、访客闭环或其他租户代码。

## 首个企业页面

入口、运行时和当前企业配置共同构成可渲染首屏：

| 路由 | 原始体积 | gzip |
| --- | ---: | ---: |
| `/c/tuotu` 首屏 JS | 322.32 kB | 98.50 kB |
| `/c/template` 首屏 JS | 317.04 kB | 96.79 kB |

相较优化前单包，`/c/tuotu` 可渲染首屏 JS 的原始体积减少约 40.7%，gzip 减少约 40.8%。其他企业配置不会随当前路由下载。
